# 名稱：團購系統

## 系統介紹
本系統可廣泛運用於多元生活場景，例如社區住戶集體採買日用品與生鮮食材，透過集中下單降低單價與運費；公司行號團購辦公用品或下午茶點心，提高採購效率並節省成本；校園學生發起零食、飲料或課程教材團購，分攤費用並提升取得便利性；以及社群團主在社群平台上發起限時團購活動，整合分散需求向廠商議價，創造更優惠的批發價格。
